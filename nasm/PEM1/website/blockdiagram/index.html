CLK : CLOCK
CLR : CLEAR

PC  : PROGRAMCOUNTER
Cp  : count programcounter
Ep  : Enable programcounter

MAR : MEMORY ADDRESS REGISTER
Lm  : Load Memory address register

PROM 16 x 8 : PROGRAMMABLE READ ONLY MEMORY 16 bytes 8 bits long
Er  : Enable ROM

IR  : INSTRUCTION REGISTER
Li  : Load instruction register
Ei  : Enable instruction register

CON : CONTROL REGISTER

A   : ACCUMULATOR REGISTER
LA  : Load accumulator
EA  : Enable accumulator

ALU : ARITHMETIC and LOGIC UNIT
SU  : Subtract flag
EU  : Enable ALU

B   : B-REGISTER
LB  : Load B-register

O   : OUTPUT REGISTER
LO  : Load output register

D   : BINARY DISPLAY
